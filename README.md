# Intrusion Detection Classification

## Project Summary
This project reproduces the methodology from the research paper 
"Enhanced Intrusion Detection Systems Performance with UNSW-NB15 Data Analysis" 
(MDPI Algorithms, 2024). We apply Logistic Regression and Decision Tree 
classification models to detect network intrusions using the UNSW-NB15 dataset.

🔗 https://www.kaggle.com/datasets/mrwellsdavid/unsw-nb15

## Project Structure
```
📁 data/        → dataset files (.csv)
📁 models/      → Jupyter notebook with all code
📄 README.md    → project summary
```

## How to Run
1. Clone this repository
2. Download the dataset from Kaggle and place it in the `data/` folder
3. Open the notebook in `models/` using Jupyter Notebook or Google Colab
4. Run all cells in order

## Models Used
- Logistic Regression
- Decision Tree (best depth tuning)

## Evaluation Metrics
Accuracy, Precision, Recall, F1-Score, Confusion Matrix
